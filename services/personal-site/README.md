# ArtDanshin - Персональный сайт

Современный персональный сайт фронтенд разработчика ArtDanshin, увлекающегося играми и стримингом.

## 🚀 Особенности

- **Современный дизайн** - Темная тема с неоновыми акцентами в стиле геймерской эстетики
- **Адаптивность** - Полностью адаптивный дизайн для всех устройств
- **Интерактивность** - Анимации, переходы и интерактивные элементы
- **Производительность** - Оптимизированный код и быстрая загрузка
- **Доступность** - Соблюдение стандартов доступности

## 🚀 Запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/artdanshin/personal-site.git
```

2. Откройте `index.html` в браузере или используйте локальный сервер:
```bash
# Python 3
python -m http.server 8000

# Node.js (если установлен http-server)
npx http-server

# PHP
php -S localhost:8000
```

3. Откройте http://localhost:8000 в браузере

## 📝 Настройка

### Изменение контента
- Отредактируйте HTML файлы для изменения текста
- Замените изображения в папке `images/`
- Обновите ссылки на социальные сети в футере

### Кастомизация стилей
- Основные переменные находятся в `styles/main.css`
- Цвета, размеры и отступы настраиваются через CSS переменные

### Добавление функциональности
- JavaScript файлы модульные и легко расширяемые
- Используйте существующие функции как основу
- Следуйте паттернам именования

## 🎨 Кастомизация

### Цвета
Измените CSS переменные в `styles/main.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #ec4899;
    --accent-color: #06b6d4;
    /* ... */
}
```

### Шрифты
Замените Google Fonts в `<head>`:
```html
<link href="https://fonts.googleapis.com/css2?family=YourFont:wght@300;400;500;600;700&display=swap" rel="stylesheet">
```

### Анимации
Настройте скорость анимаций через CSS переменные:
```css
:root {
    --transition-fast: 0.15s ease-in-out;
    --transition-normal: 0.3s ease-in-out;
    --transition-slow: 0.5s ease-in-out;
}
```
