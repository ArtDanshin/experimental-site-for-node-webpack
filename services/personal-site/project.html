Обзор проекта

Персональный сайт ArtDanshin - это современный портфолио, созданный с использованием чистого HTML, CSS и JavaScript. Сайт демонстрирует навыки фронтенд разработки и служит платформой для размещения проектов, статей и контактной информации.

Основной задачей было создать сайт, который бы отражал увлечения разработчика играми и стримингом, при этом оставаясь профессиональным и функциональным. Дизайн выполнен в темной геймерской тематике с неоновыми акцентами.

Вызовы и решения

Создание адаптивного дизайна

Основной вызов заключался в создании дизайна, который бы одинаково хорошо выглядел на всех устройствах. Решение: использование CSS Grid и Flexbox с mobile-first подходом.

Оптимизация производительности

Необходимо было обеспечить быструю загрузку сайта. Решение: оптимизация CSS, минификация кода и ленивая загрузка изображений.

Создание интерактивности

Добавление анимаций и интерактивных элементов без использования тяжелых библиотек. Решение: использование чистого JavaScript и CSS анимаций.
